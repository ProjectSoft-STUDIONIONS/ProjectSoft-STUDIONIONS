# ProjectSoft

* [module-pdf-metadata](https://github.com/ProjectSoft-STUDIONIONS/module-pdf-metadata) - Модуль EvolutionCMS для редактирования метаданных PDF файла
* [evoFileManagerDialog](https://github.com/ProjectSoft-STUDIONIONS/evoFileManagerDialog) - Плагин открытия файлменеджера не в новом окне браузера, а во всплывающем окне админки EvolutionCMS.
* [food-module](https://github.com/ProjectSoft-STUDIONIONS/food-module) - Модуль FoodModule для загрузки файлов xlsx или pdf в папку `/food/`. Модуль и плагин для модуля на EvolutionCMS. Модуль актуален для сайтов школ России.
* [food-uploader-plugin](https://github.com/ProjectSoft-STUDIONIONS/food-uploader-plugin) - WordPress Плагин для загрузки файлов xlsx или pdf в папку `/food/`, доступен только администраторам. Плагин актуален для сайтов школ России.
* [com_food](https://github.com/ProjectSoft-STUDIONIONS/com_food) - Компонент «ПИТАНИЕ» Joomla CMS для загрузки файлов xlsx или pdf в папку `/food/` Компонент актуален для сайтов школ России.
* [FontMassivePackInstaller](https://github.com/ProjectSoft-STUDIONIONS/FontMassivePackInstaller) - Набор программ для работы со шрифтами. Официальная страница FontMassive Pack - [https://fontmassive.com/](https://fontmassive.com/)
* [EvolutionCMS-Utilites](https://github.com/ProjectSoft-STUDIONIONS/EvolutionCMS-Utilites) - Набор плагинов для EvolutionCMS
* [tinymce-codemirror-evolutioncms](https://github.com/ProjectSoft-STUDIONIONS/tinymce-codemirror-evolutioncms) - Плагин исходного кода Codemirror для TinyMCE >= 4.x под EvolutionCMS
* [NotoColorEmoji-TinyMCE](https://github.com/ProjectSoft-STUDIONIONS/NotoColorEmoji-TinyMCE) - Плагин подключения Emoji в редактор TinyMCE >= 4.x административной пенели EvolutionCMS
* [scan-images-menu2pdf](https://github.com/ProjectSoft-STUDIONIONS/scan-images-menu2pdf) - Конвертирование отсканированных изображений меню в PDF файлы. Обрабатывается два типа меню. Каждодневное и Двухнедельное. Актуально для школ России.
* [radioApp-lite](https://github.com/ProjectSoft-STUDIONIONS/radioApp-lite) - Приложение nwjs «Ваше Радио» (облегчённая версия). 
* [space-lively-wallpaper](https://github.com/ProjectSoft-STUDIONIONS/space-lively-wallpaper) - Анимированные обои «Сквозь вселенную» для [Lively Wallpaper](https://github.com/rocksdanister/lively)
